How it works

Fully serverless. Your files never touch an app server.

01
⬆️

Direct upload to S3

Your browser uploads files directly to S3 via presigned URLs — bypassing any server bottleneck.

→
02
⚙️

Lambda + FFmpeg

An AWS Lambda (Go runtime, Docker) converts each image in parallel goroutines using FFmpeg.

→
03
📡

Real-time notification

AWS IoT Core pushes an MQTT event to your browser the instant the ZIP is ready.

→
04
⬇️

Download & gone

A presigned download URL is delivered. Files are wiped from S3 after 24 hours.
